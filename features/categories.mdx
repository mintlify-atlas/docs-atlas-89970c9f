## Overview

Por el Barrio uses a flexible category system (`tipos_sitio`) to organize places into logical groups. Categories are dynamically loaded from Supabase and displayed as filterable chips.

## Database Schema

Categories are stored in the `tipos_sitio` table:

```sql
CREATE TABLE tipos_sitio (
  id INT PRIMARY KEY AUTO_INCREMENT,
  tipo VARCHAR(100) NOT NULL,
  descripcion TEXT
);
```

**Fields:**
- `id`: Unique identifier
- `tipo`: Category name (e.g., "Restaurant", "Shop", "Service")
- `descripcion`: Optional description of the category

## Data Hook

### useTiposSitio

The `useTiposSitio` hook fetches all categories:

```tsx
import { useTiposSitio } from "@/hooks/use-tipos-sitio";

export function useTiposSitio() {
  const [tipos, setTipos] = useState<TipoSitio[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  const fetchTipos = useCallback(async () => {
    const { data, error: err } = await supabase
      .from("tipos_sitio")
      .select("id, tipo, descripcion")
      .order("tipo");
    
    if (err) throw err;
    setTipos(data as TipoSitio[]);
  }, []);

  useEffect(() => {
    fetchTipos();
  }, [fetchTipos]);

  return { tipos, loading, error, refresh: fetchTipos };
}
```

**TypeScript Type:**

```tsx
export type TipoSitio = {
  id: number;
  tipo: string;
  descripcion: string | null;
};
```

## Filtering Logic

### Available Categories

Only categories that have places in the user's province are shown:

```tsx
const tiposDisponibles = useMemo(() => {
  // Get all category IDs present in filtered places
  const idsPresentes = new Set(
    sitiosPorProvincia
      .map((s) => s.tipo_sitio_id)
      .filter((id): id is number => id != null)
  );
  
  // Filter tipos to only those with places
  return tipos.filter((t) => idsPresentes.has(t.id));
}, [sitiosPorProvincia, tipos]);
```

### Category Counts

Calculate the number of places in each category:

```tsx
const countsByTipoId = useMemo(() => {
  const counts: Record<number, number> = {};
  
  for (const sitio of sitiosPorProvincia) {
    const tipoId = sitio.tipo_sitio_id;
    if (tipoId == null) continue;
    counts[tipoId] = (counts[tipoId] ?? 0) + 1;
  }
  
  return counts;
}, [sitiosPorProvincia]);
```

### Applying Filter

```tsx
const sitiosFiltrados = useMemo(() => {
  let filtrados = sitiosPorProvincia;
  
  if (selectedTipoId !== null) {
    filtrados = filtrados.filter((s) => s.tipo_sitio_id === selectedTipoId);
  }
  
  return filtrados;
}, [sitiosPorProvincia, selectedTipoId]);
```

## Category Selection

### State Management

Category selection is managed with local state and URL params:

```tsx
const [selectedTipoId, setSelectedTipoId] = useState<number | null>(null);
const params = useLocalSearchParams<{ categoriaId?: string }>();

// Apply filter from URL params (from drawer)
useEffect(() => {
  if (params.categoriaId) {
    const categoriaId = parseInt(params.categoriaId, 10);
    if (!isNaN(categoriaId)) {
      setSelectedTipoId(categoriaId);
    }
  }
}, [params.categoriaId]);

// Update URL when selection changes
const handleSelectTipo = useCallback((tipoId: number | null) => {
  setSelectedTipoId(tipoId);
  
  if (tipoId !== null) {
    router.setParams({ categoriaId: String(tipoId) });
  } else {
    router.setParams({});
  }
}, []);
```

### Integration with Drawer

The drawer menu can set the category filter by navigating with params:

```tsx
// In drawer menu
<TouchableOpacity
  onPress={() => {
    router.push({
      pathname: "/(drawer)/(tabs)/",
      params: { categoriaId: String(tipo.id) }
    });
  }}
>
  <Text>{tipo.tipo}</Text>
</TouchableOpacity>
```

## Header Integration

The active category is displayed in the header when the category bar is hidden:

```tsx
const { setHeaderChip, setHeaderCategoryLabel } = useHeaderCategory();

const selectedCategoryLabel = useMemo(() => {
  if (selectedTipoId == null) return "Todas";
  const tipo = tipos.find((t) => t.id === selectedTipoId);
  return tipo?.tipo ?? "Todas";
}, [selectedTipoId, tipos]);

useEffect(() => {
  setHeaderCategoryLabel(selectedCategoryLabel);
}, [selectedCategoryLabel, setHeaderCategoryLabel]);
```
