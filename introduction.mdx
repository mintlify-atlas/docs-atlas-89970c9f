## Welcome to Por el Barrio

Por el Barrio is a mobile-first application that connects people with local businesses and places in their neighborhood. Built with React Native and Expo, it provides a seamless experience for discovering, reviewing, and sharing local spots.

## Tech Stack

- **Frontend**: React Native, Expo, NativeWind (Tailwind CSS)
- **Backend**: tRPC, Express.js
- **Database**: Supabase (PostgreSQL)
- **State Management**: TanStack Query, React Context
- **Navigation**: Expo Router (file-based routing)

## Key Features

  <Accordion title="Location-based Discovery">

## Getting Started

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install all required packages
    ```bash
    pnpm install
    ```
  </Step>
  <Step title="Configure Environment">
    Set up your `.env` file with Supabase credentials
    ```bash
    cp .env.example .env
    ```
  </Step>
  <Step title="Run Development Server">
    Start both the API server and Expo Metro bundler
    ```bash
    pnpm dev
    ```
  </Step>
</Steps>
