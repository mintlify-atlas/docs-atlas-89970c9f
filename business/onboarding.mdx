## Overview

Por el Barrio provides a simple, WhatsApp-based onboarding process for local business owners who want to list their businesses in the app. This approach makes it accessible to owners who may not be tech-savvy.

## Onboarding Flow

The onboarding process is handled through the **Colaboración** tab (`app/(drawer)/(tabs)/colaboracion.tsx`):

<Steps>
  <Step title="Download Request Form">
    Business owner downloads an Excel form with all required fields.
  </Step>
  
  <Step title="Fill Out Form">
    Owner completes the form with their business information.
  </Step>
  
  <Step title="Send via WhatsApp">
    Owner sends the completed form to the developer via WhatsApp.
  </Step>
  
  <Step title="Review & Approval">
    Developer reviews the submission and adds the business to the database.
  </Step>
</Steps>

## Screen Implementation

### Configuration

```tsx
const URL_SOLICITUD =
  "https://djxvosobflwtryenqoou.supabase.co/storage/v1/object/public/descargas/Solicitud.xlsx";
const WHATSAPP_NUMERO = "5352708602";
const URL_WHATSAPP = `https://wa.me/${WHATSAPP_NUMERO}`;
```

**Configuration Values:**
- `URL_SOLICITUD`: Direct download link for the Excel form (hosted on Supabase Storage)
- `WHATSAPP_NUMERO`: Developer's WhatsApp number (international format, no +)
- `URL_WHATSAPP`: WhatsApp API deep link

## Request Form

### Form Structure

The Excel form (`Solicitud.xlsx`) includes these fields:

<Tabs>
  <Tab title="Basic Info">
    - **Nombre del negocio** (Business name)
    - **Descripción** (Description)
    - **Tipo de sitio** (Category: Restaurant, Shop, Service, etc.)
  </Tab>
  
  <Tab title="Contact">
    - **Teléfono** (Phone number)
    - **Email** (Optional)
    - **Dirección** (Physical address)
  </Tab>
  
  <Tab title="Location">
    - **Provincia** (Province)
    - **Municipio** (Municipality)
    - **Localización** (GPS coordinates: latitude,longitude)
  </Tab>
  
  <Tab title="Media">
    - **Imágenes** (Image URLs, comma-separated)
    - **Ofertas** (Special offers or promotions)
    - **Menús** (Menu information, if applicable)
  </Tab>
</Tabs>

### Form Template

The form is stored in Supabase Storage:

```
https://<project>.supabase.co/storage/v1/object/public/descargas/Solicitud.xlsx
```

<Info>
  You'll need to upload the Excel template to your Supabase Storage bucket named `descargas` with public access.
</Info>

## WhatsApp Integration

### Deep Link Format

The app uses the WhatsApp URL scheme to open a conversation:

```tsx
const WHATSAPP_NUMERO = "5352708602"; // No + symbol
const mensaje = "Hola, me gustaria que mi negocio...";
const mensajeCodificado = encodeURIComponent(mensaje);
const url = `https://wa.me/${WHATSAPP_NUMERO}?text=${mensajeCodificado}`;
```

**URL Format:**
```
https://wa.me/5352708602?text=Hola%2C%20me%20gustaria...
```

### Platform Behavior

<Tabs>
  <Tab title="iOS">
    Opens WhatsApp app if installed, otherwise opens WhatsApp Web in Safari.
  </Tab>
  
  <Tab title="Android">
    Opens WhatsApp app if installed, otherwise prompts to install or opens WhatsApp Web.
  </Tab>
  
  <Tab title="Web">
    Opens WhatsApp Web in a new tab.
  </Tab>
</Tabs>

### Pre-filled Message

The WhatsApp message is pre-filled with:

```
Hola, me gustaria que mi negocio o emprendimiento aparezca en la aplicación. Aqui envio el formulario
```

The user can attach the Excel file manually in WhatsApp.

## Developer Workflow

When the developer receives a submission:

<Steps>
  <Step title="Receive Form">
    Form arrives via WhatsApp with business information.
  </Step>
  
  <Step title="Validate Data">
    - Verify all required fields are complete
    - Check GPS coordinates are valid
    - Ensure images are accessible
    - Confirm province/municipality exist in database
  </Step>
  
  <Step title="Add to Database">
    Insert into `sitios_relevantes` table via Supabase dashboard or SQL:
    
    ```sql
    INSERT INTO sitios_relevantes (
      nombre, descripcion, tipo_sitio_id, telefono,
      direccion, provincia_id, municipio_id, localizacion,
      imagenes, ofertas, menus
    ) VALUES (
      'El Buen Sabor',
      'Restaurante de comida criolla',
      1, -- Restaurant category
      '53123456',
      'Calle 23 #456, Vedado',
      '1', -- La Habana
      '101', -- Plaza de la Revolución
      '23.13194,-82.38304',
      'https://example.com/image1.jpg,https://example.com/image2.jpg',
      'Descuento 10% en almuerzos',
      NULL
    );
    ```
  </Step>
  
  <Step title="Notify Owner">
    Send confirmation message via WhatsApp that the business has been added.
  </Step>
</Steps>
